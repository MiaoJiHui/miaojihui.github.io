hexo的使用

Jun 30, 2016

已经拖了好久了，终于下定决心想把自己怎么使用hexo更新博客的过程记录下来，互勉

首先你得了解如下技术和技能：

  • npm
    –node package management node包管理，具体可以百度，总之是开发神器
  • hexo
    –这是一款开源的博客模板引擎，我就是用的它生成的这个博客，与之对应的还有jekyll等
  • git
    –开源的分布式版本控制系统，可以方便的在本地/服务器更新和提交代码，比较小巧高效灵活
  • markdown
    –按我的理解就是一种文本输入方式
  • github
    –这是广大程序员的代码分享之地，可以托管自己的代码也可以fork他人的代码，还可以建一个简单的github博客
